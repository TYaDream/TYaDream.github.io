Linux操作系统和常用命令

发表于 2022-10-25 更新于 2022-10-29 分类于 Linux

Linux文件系统



常用命令

ls 路径 ：看路径的目录

ls :看当前目录

ls 路径 -a：顺便看到隐藏内容
ls 路径 -l ：看到文件详细信息
ls 路径 -la：结合上面俩

cd: 目录切换


如 cd / :就是到根目录
cd ./ :就是到当前目录

clear清空屏幕

pwd查看当前路径

小技巧

mkdir：创建文件夹

mkdir aaa :在当前目录创建aaa文件夹
mkdir bbb/ccc :在bbb创建ccc，如果bbb不存在则不能创建
mkdir bbb/ccc -p :在bbb创建ccc，如果bbb不存在则创建bbb和ccc

rm：删除


如 rm -f bbb

ll：查看文件夹内所有文件的详细内容

lll aaa/ ：可以看见该目录下所有文件的详细内容

cp：复制

cp aaa bbb/ 把aaa复制到bbb里面
cp ccc/ bbb/ 报错
cp -r ccc/ bbb/ 把ccc里的东西复制到bbb里（递归复制）

mv ： 移动文件

cat，more，tail：文件内容查看

data和cal：日期，日历

显示当前日期或日历

echo：控制台输出


如echo 1111
控制台将会输出1111

|（管道命令）


如ps -ef | grep sshd

ps进程查看，kill关闭进程

ps -ef 显示目前Linux的进程

kill -9 进程号：关闭进程

grep sshd 在进程中寻找sshd服务

> 和>>（重定向）


如echo hello > 1.txt
1.txt的内容变成hello了

tar（解压缩）

压缩

解压缩

不加-C是不行的，不加的话意思是解压到当前目录下
指定压缩算法

root ：超级用户

进入超级用户

内存，磁盘利用率查看free，df


图上第一个命令是free -h
